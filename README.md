# nao_retarget_motion

Peng, Xue & Coumans, Erwin & Zhang, Tingnan & Lee, Tsang-Wei & Tan, Jie & Levine, Sergey. (2020). Learning Agile Robotic Locomotion Skills by Imitating Animals.https://arxiv.org/pdf/2004.00784.pdf
